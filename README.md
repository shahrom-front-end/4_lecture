# 1 Метод pop() удаляет последний элемент из массива и возвращает его значение
```js
    let num=[1,2,3,4,5,6]
    let num2=num.pop()
    console.log(num2);// 6
    console.log(num);// [ 1, 2, 3, 4, 5 ]
```
# 2 Метод shift() удаляет первый элемент из массива и возвращает его значение. Этот метод изменяет длину массива.
```js
    let num=[1,2,3,4,5,6]
    let num2=num.shift()
    console.log(num2);// 1
    console.log(num);// [ 2, 3, 4, 5, 6 ]
``` 
# 3 Метод push() добавляет один или более элементов в конец массива и возвращает новую длину массива
```js
    let num2=num.push(123)
    console.log(num2);// 7
    console.log(num);// [ 2, 3, 4, 5, 6, 123 ]
```
# 4 Метод unshift() добавляет один или более элементов в начало массива и возвращает новую длину массива
```js
    let num=[1,2,3,4,5,6]
    let num2=num.unshift(123)
    console.log(num2);// 7
    console.log(num);// [ 123,1, 2, 3, 4, 5, 6 ]
```

# 5 Метод concat() возвращает новый массив, состоящий из массива, на котором он был вызван, соединённого с другими массивами и/или значениями, переданными в качестве аргументов
```js
    let num=[1,2]
    let num2=["food",12,"moon"]
    console.log(num.concat(num2));// [ 1, 2, 'food', 12, 'moon' ]
    console.log(num);// [ 1, 2 ]
```
# 6 Метод slice() возвращает новый массив, содержащий копию части исходного массива
```js
    let num=[1,2,3,4,5,6]
    let num2=["food",12,"moon"]
    console.log(num.slice(1,5));//[ 2, 3, 4, 5 ]
    console.log(num);// [ 1, 2, 3, 4, 5, 6 ]
```

# 8 Метод includes() определяет, содержит ли массив определённый элемент, возвращая в зависимости от этого true или false 
```js
    let num="shahrom"
```

# 9 Метод indexOf() возвращает индекс первого вхождения указанного значения в строковый объект String , на котором он был вызван, начиная с индекса fromIndex 
```js
    let num="shahrom"
    console.log(num.indexOf("rom"));//4
```
# 10 Метод splice() изменяет содержимое массива, удаляя существующие элементы и/или добавляя новые
```js
    let num = ["shahrom", 123, "nabi", "umed", "navrus"]
    let num2=num.splice(1, 1, "dars bxoonn")
    console.log(num2);//[ 123 ]
    console.log(num);//[ 'shahrom', 'dars bxoonn', 'nabi', 'umed', 'navrus' ]
```
# 11 Метод toString() преобразует объект в строковое представление. Метод автоматически вызывается JavaScript, когда объект нужно представить в текстовом виде. Если метод не переопределён, то он возвращает строку формата [object тип] , где тип — это строка, которая уточняет тип объекта
```js
    let num=[1,2,3]
    let num2=num.toString()
    console.log(typeof(num2));//string
    console.log(num2);//1,2,3
```
# 12 Метод toReversed()считывает lengthсвойство this. Затем он посещает каждый индекс между length - 1и 0в порядке убывания и добавляет значение этого индекса в исходном массиве к соответствующему индексу в новом массиве
```js
    let num=[1,2,3]
    console.log(num.toReversed())//[3,2,1]
```
# 13 Метод forEach() выполняет функцию callback один раз для каждого элемента, находящегося в массиве в порядке возрастания. Она не будет вызвана для удалённых или пропущенных элементов массива. Однако, она будет вызвана для элементов, которые присутствуют в массиве и имеют значение undefined

# 14 Метод map() позволяет трансформировать один массив в другой при помощи функций-колбэка. Переданная функция будет вызвана для каждого элемента массива по порядку. Из результатов вызова функции будет собран новый массив
```js
    let num=[1,2,3]
    let cap=num.map(Math.sqrt)
    console.log(cap.toReversed())//[1.7320508075688772, 1.4142135623730951, 1]
```
# 15 Метод find() возвращает значение первого найденного в массиве элемента, которое удовлетворяет условию переданному в callback функции. В противном случае возвращается undefined . Также смотрите метод findIndex() , который возвращает индекс найденного в массиве элемента вместо его значения   
```js
    let num=[1,2,3]
    let cap=num.find((e)=>{
        return e>2
    })
    console.log(cap)//3
```
# 16 Метод filter() создаёт новый массив со всеми элементами, прошедшими проверку, задаваемую в передаваемой функции
```js
    let num=[1,2,3]
    let cap=num.filter((e)=>{
        return e>2
    })
    console.log(cap)//[3]
```
# 17 Метод массива reduce() позволяет превратить массив в любое другое значение с помощью переданной функции-колбэка и начального значения. Функция-колбэк будет вызвана для каждого элемента массива, и всегда должна возвращать результат.
```js
    let num=[1,2,3]
    let cap=num.reduce((e)=>{
        return e>2
    })
    console.log(cap)//false
```
# 18 Метод toSorted()считывает lengthсвойство this. Затем он собирает все существующие свойства с целыми ключами в диапазоне от 0до length - 1, сортирует их и записывает в новый массив.
```js
    let num=[1,2,3]
    let cap=num.toSorted((e)=>{
        return e-1
    })
    console.log(cap)//[1,2,3]
```

## SPREAAD---Spread оператор – это три точки перед именем переменной (или константы), внутри определения объекта. Он раскладывает соответствующий объект внутри нового объекта. С его помощью можно получить только копию, он не может изменять существующие объекты.
